# Beta Outreach

Keep outreach direct and honest. This is an early beta, so ask people to test specific flows instead of just saying "try it."

## Feedback Form

Create a Google Form or Tally form and paste the live link here before inviting testers:

```text
Feedback form: <paste link here>
Live app: <paste frontend URL here>
Known issues: <paste short notes here>
```

## Friend / Family / Instagram DM

```text
Hey, I am testing an early beta of my Vedic compatibility dating app.

Could you create a profile, verify your email, browse a few profiles, try liking/requesting compatibility, and send me feedback through this form?

Live app: <link>
Feedback form: <link>

It is still beta, so rough edges are expected. The most helpful feedback is anything that breaks, feels confusing, or looks off on your phone.
```

## Tester Task List

```text
Please try these if you have time:
1. Create a profile with photos.
2. Verify your email.
3. Browse profiles.
4. Like someone or request compatibility.
5. Try chat if you get a match.
6. Test report/block only if you are comfortable.
7. Submit the feedback form with your device/browser.
```

```text
The app lets people create a profile, generate chart-based compatibility, request deeper readings, match, and chat. It is still early, so I am looking for testers who are willing to try the full flow and give honest feedback.

If you are open to testing:

- verify your email
- browse/match/request compatibility
- test it on mobile if possible
- submit feedback through the form

Live app: <link>
Feedback form: <link>

```

## Bug Log Format

Use this shape in a notes doc or spreadsheet:

```text
Severity: blocker | major | minor | polish
Tester:
Device/browser:
Account email:
Flow:
What happened:
Expected:
Screenshot/link:
Status: open | fixed | won't fix before beta
```
